# JPEG
The aim of this software is to provide an extension of the JPEG compression that considers 8N x 8N pixel box (N is an integer chosen by the user) instead of 8 X 8 
pixel box.

This software is developed by using the following Python library:
1) NumPy 
2) Scipy
3) PIL - Python Image Library
4) PiQt4
